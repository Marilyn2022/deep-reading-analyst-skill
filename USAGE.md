# Deep Reading Analyst Skill - 使用指南

## 📖 简介

这是一个专业的深度阅读分析skill，帮助你将表面阅读转化为深度学习。通过结构化思维、批判性思维、第一性原理、系统思维和六顶思考帽等多种思维模型，系统性地分析文章、论文和长文内容。

## 🎯 核心功能

1. **结构化理解** - 快速提取文章核心结构和关键信息
2. **批判性分析** - 识别论证缺陷和逻辑漏洞
3. **第一性原理** - 剥离假设，回归本质
4. **系统思维** - 建立知识关联和全局视角
5. **六顶思考帽** - 多角度全面评估
6. **实践转化** - 将知识转化为可执行的行动方案

## 📥 安装方法

### 方式一：Claude桌面端
1. 下载 `deep-reading-analyst.skill` 文件
2. 打开Claude桌面应用
3. 点击右上角的头像
4. 选择「Settings」→「Skills」
5. 点击「Import Skill」
6. 选择下载的 .skill 文件

### 方式二：Claude网页版
1. 访问 claude.ai
2. 点击右下角的齿轮图标（Settings）
3. 选择「Skills」标签页
4. 点击「Import」按钮
5. 上传 .skill 文件

## 🚀 使用方式

### 基础用法

直接将文章内容或链接发送给Claude，并说明你的需求：

```
帮我深度分析这篇文章：[粘贴文章内容或URL]
```

或者更具体地说：

```
请用批判性思维分析这篇文章的论证质量
请用第一性原理拆解这个概念
请用系统思维分析这些要素之间的关系
请用六顶思考帽全面评估这个观点
```

### 指定分析深度

你可以选择四个分析级别：

- **Level 1 (快速)** - 15分钟，适合快速了解
- **Level 2 (标准)** - 30分钟，适合日常学习
- **Level 3 (深度)** - 60分钟，适合重要内容
- **Level 4 (研究)** - 120分钟+，适合学术研究

示例：
```
请用Level 3深度模式分析这篇关于软件工程的文章
```

### 指定输出格式

你可以要求特定的输出格式：

```
请分析这篇文章并输出：
1. 学习笔记（给自己看的）
2. 分享文案（给别人看的）
3. 实践应用计划
4. 概念地图
```

```
我想系统学习"需求分析"这个主题，这是第一篇文章：

请用系统思维帮我：
- 建立这个领域的知识图谱
- 找出与其他领域的关联

```

## 🎨 技巧和最佳实践

### ✅ 推荐做法

1. **明确目的** - 告诉Claude你阅读的目的（学习/解决问题/写作参考）
2. **分段消化** - 对于超长文章，可以先分析部分章节
3. **主动提问** - 在分析过程中主动提出疑问
4. **实践验证** - 要求生成可执行的行动计划
5. **建立连接** - 要求关联到你已知的知识

### ❌ 避免做法

1. 不要一次性分析过多文章（建议≤3篇）
2. 不要期望skill代替阅读（它是辅助工具）
3. 不要忽视skill提出的批判性问题
4. 不要只收集笔记不实践

## 🆘 常见问题

**Q: Skill能否直接读取网页链接？**
A: 可以！Claude可以使用web_fetch工具获取网页内容。

**Q: 分析一篇文章需要多久？**
A: 取决于深度级别，从15分钟（快速）到120分钟（研究级）不等。

**Q: 我可以要求只使用某个思维模型吗？**
A: 当然可以！明确告诉Claude你想用哪个模型即可。

**Q: Skill能分析非英文内容吗？**
A: 可以！支持中文、英文等多种语言。

**Q: 如何让分析更符合我的需求？**
A: 提供更多上下文信息，如你的背景、阅读目的、关注重点等。
