# Deep Reading Analyst - Claude AI 深度阅读技能

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/yourusername/deep-reading-analyst)

一个专业的Claude AI技能包，通过系统化的思维框架将表面阅读转化为深度学习。

[English](README.md) | [使用指南](USAGE.md)

## 🌟 核心特性

通过以下功能提升阅读体验：

- **结构化理解** - 快速提取核心结构和关键信息
- **批判性分析** - 识别逻辑漏洞和论证缺陷
- **第一性原理** - 剥离假设，回归本质
- **系统思维** - 建立知识关联，洞察全局
- **六顶思考帽** - 多角度全面评估
- **实践转化** - 将知识转化为可执行计划

## 🎯 技能功能

本技能提供系统化的5步分析流程：

1. **初始化** - 设定目标和深度级别
2. **结构理解** - 提取文章结构和核心概念
3. **思维模型** - 运用多种框架（4个深度级别可选）
4. **综合输出** - 基于目标生成定制化内容
5. **知识激活** - 创建可执行的下一步行动

## 📊 分析深度级别

根据需求选择合适的级别：

| 级别 | 时间 | 适用场景 |
|------|------|----------|
| **Level 1 (快速)** | 15分钟 | 快速了解核心观点 |
| **Level 2 (标准)** | 30分钟 | 日常学习和理解 |
| **Level 3 (深度)** | 60分钟 | 重要内容深度分析 |
| **Level 4 (研究)** | 120分钟+ | 学术研究和跨源对比 |

## 🚀 安装方法

### 前置要求

- Claude桌面应用或Claude网页版 (claude.ai)
- Claude Pro订阅（推荐，以获得最佳体验）

### 安装步骤

1. **下载**技能文件：[deep-reading-analyst.skill](deep-reading-analyst.skill)

2. **导入到Claude：**
   
   **桌面应用：**
   - 打开Claude桌面应用
   - 点击右上角头像
   - 选择 Settings → Skills
   - 点击 "Import Skill"
   - 选择下载的 `.skill` 文件

   **网页版：**
   - 访问 [claude.ai](https://claude.ai)
   - 点击右下角设置图标
   - 进入 "Skills" 标签页
   - 点击 "Import" 按钮
   - 上传 `.skill` 文件

3. **开始使用！** 当你要求Claude分析文章时，技能会自动触发。

## 💡 快速开始

### 基础用法

直接提供文章并要求分析：

```
请深度分析这篇文章：[粘贴内容或URL]
```

### 指定分析深度

```
请用Level 3（深度）模式分析这篇关于软件工程的文章
```

### 请求特定输出格式

```
请分析这篇文章并提供：
1. 学习笔记（给自己看的）
2. 实践应用计划
3. 概念地图
```

## 📚 内置思维框架

技能包含详细的参考文档：

- **批判性思维** - 论证评估、逻辑谬误检测、证据分析
- **第一性原理** - 假设识别、本质提取、从零重建
- **系统思维** - 因果循环、杠杆点、跨领域模式
- **六顶思考帽** - 白帽（事实）、红帽（感受）、黑帽（风险）、黄帽（价值）、绿帽（创新）、蓝帽（流程）
- **费曼技巧** - 简化和基于教学的理解
- **多种输出模板** - 8种不同格式满足各种需求
